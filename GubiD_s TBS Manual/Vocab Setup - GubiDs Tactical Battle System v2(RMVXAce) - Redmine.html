Vocab Setup¶

How to change vocabulary inside GTBS ?

Don't touch %d ; %s ; %1$s symbols.
Don't remove the ' or the " marks, just change inside those marks.

1st step :

Open the : "[GTBS] Interface Settings" script and change:

Line 005 VICTORY_MESSAGE = "CHANGE_HERE"
Line 006 ESCAPE_MESSAGE = "CHANGE_HERE"
Line 007 DEFEAT_MESSAGE = "CHANGE_HERE"
Line 008 BATTLE_COMPLETE = "CHANGE_HERE"
Line 009 ITEM_GAIN_TEXT = "CHANGE_HERE"

Line 012 BATTLE_START = "- CHANGE_HERE -"

Line 017 AT_TERM = "CHANGE_HERE"

2sd step :
Then, open the "[GTBS] Vocab Setup" script and change:

Line 003 Place_Message = "CHANGE_HERE"

3rd step :
Then open the "[GTBS] Info Sprites" and replace the (line 468):

def create_bitmap
if FileTest.exist?(sprintf("Graphics/Pictures/GTBS/%s_Turn.png", @turn))
bmp = Cache.picture(sprintf("GTBS/%s_Turn.png", @turn))
else
if @turn == Scene_Battle_TBS::Turn_Player
team = Vocab_GTBS::Players
elsif @turn == Scene_Battle_TBS::Turn_Enemy
team = Vocab_GTBS::Enemies
end
bmp = Bitmap.new(300,180)
bmp.font.name = GTBS::font
bmp.font.size = 60
bmp.font.color = Color.new(30,30,30,255)
bmp.draw_text(2,2,300,180,sprintf("%s Turn", team),1)
bmp.font.color = Color.new(255,255,255,255)
bmp.draw_text(0,0,300,180,sprintf("%s Turn", team),1)
end
self.bitmap = bmp
end
