Actor Config¶

Put these into actors' noteboxes.

General Tags¶

Large Actor Units¶

large=size

size: numerical; actor will take up size^2 tiles on the map, and be unable to pass through smaller spaces.

Death Animation¶

death_anim=anim_id

anim_id: numerical; ID of animation in database to be played on actor death.

Mini Scene Battler¶

no_mini

This is used if you want an actor to use their in battle sprite as the graphic for Mini-Scenes.

Actor AI Tags¶

Actor AI¶

tactics=AI

AI: string; name of an AI set up in AI Setup

Sets the AI used by the actor, if AI-controlled.

Prevent Actor Wander¶

prevent_wander

Prevents AI-controlled actors from wandering without targets in sight range. View-range for actors is determined by class, and can be set in either the class' Class Config or in the Actor and Class Setup script.

Summon Tags¶

Setting Actor as Summon¶

summon

Designates an actor as a summon, rather than a regular character, and subject to the usual summon restrictions.

Setting Summons Controllable¶

controllable

Designates a summon as player-controllable. If not set, summon will be AI-controlled.

Number of Instances Allowed¶

allowed=number

number: integer; the number of copies of a summon allowed on a given map.

Summon Doom Turn¶

doom_turn=turns

turns: integer; the number of turns the summon can remain on the field for. Only used if 'DOOM_SUMMONS = true' in Skill Setup

Traverse Type:¶

traverse=TYPE

TYPE: number; Used to determine which type of terrain tags affect this unit. Valid values are:
  • 0 : Normal (Affected normally by all terrains)(default if not defined)
  • 1 : Thick (Not affected by thick terrain (assumed Terrain Tag 1))
  • 2 : Rocky (Not affected by mountainous(rocky) terrain. (Assumed Terrain Tag 2))
  • 3 : Mounted (Not affected by thick or rocky terrain)
