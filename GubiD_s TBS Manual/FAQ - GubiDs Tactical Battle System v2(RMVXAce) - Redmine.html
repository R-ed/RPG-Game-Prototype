FAQ¶

* How do I use the engine?¶

If it is the first time you are using GTBS, please take a look to the Documentation Area.
If there is something you didn't understood in the documentation Area, then use the search bar on this website, you are probably not the first one :)
Of course, if you didn't find your answer, feel free to open a topic on the forum.
If you founded a bug (Not already found) feel free to open a bug ticket.

* I encounter an error, what should I do?¶

First, you should search in the "issues" part if someone have the same issue.
-> If not, try to duplicate it from the demo of GTBS to check if they are not any compatibility script issue.
If you can replicate it from the GTBS Demo, then explain how you did and send the "error log" to GubiD.
If it's happen only in your project, try to produce a little demo, and explain when the error pop out, then send that with the error log to GubiD.

* How do I know the actors/weapons/enemies ID?¶

* Why are my note tags not working?¶

All tags must be exactly as they are in the script. Lowercase. For a list of commands available see the appropriate object section or the Note_Config section for more details.

* When I set up an actor/enemy they are invisible, or I get a cannot find graphic message.¶

The system will automatically take whatever image is assigned to the enemy in the database and add a $ to it. This is done so that the default battle system and the GTBS can work accordingly. Without having to specify otherwise what the name should be.
Actors will use the current character set name and index to load the SAME FILE from the BATTLERS folder rather than CHARACTERS. It does this because many people want a alternate graphic for battles as compared to regular map. Character Set that are animated are no longer (as in previous versions of gtbs) by the character set filename. It is not controlled by setting a flag on the actor in the note box. See the Note_Config section for more details.

* Can I use GTBS with another custom battle system?¶

YES! Just run 'disable_tbs' or 'enable_tbs' when you want to enable/disable the TBS.

* What scripts can be used alongside GTBS ace?¶

It depends on what it affects. Most scripts are compatible, but those that affect battle flow are likely to NOT work. There are a few that have been tested already, such as Tsukihime's Mana shield, but not all are guaranteed to work.
