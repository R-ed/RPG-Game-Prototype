Interface Settings¶

String Values
Option Name Default Description
VICTORY_MESSAGE "Congratulations!" Message displayed on victory.
ESCAPE_MESSAGE "Escape!" Message displayed on escape.
DEFEAT_MESSAGE "You was defeated!" Message displayed on being defeated
BATTLE_COMPLETE "Battle Complete" Message displayed when a battle is completed without dying.
ITEM_GAIN_TEXT "Items Found" Text displayed when items are found.
BATTLE_START "-Battle Start-" If no picture named 'Battle_Start.png' is present in 'Graphics/Pictures/', this text will be displayed at the beginning of battles.
AT_TERM "AT" Active Time gauge abbreviation.
Unknown_Text "????" Displayed instead of HP and MP values if the enemy is tagged with unknown HP and MP.
THINKING_TEXT "Thinking" Text displayed whilst the enemy is delaying prior to making its moves.
HP_Status_Img "" If Use_Dual_Gradient_Bars is false, specifies the image to be used to fill the HP bar. If still unspecified, the gauge will use two-tone gradiation.
MP_Status_Img "" If Use_Dual_Gradient_Bars is false, specifies the image to be used to fill the MP bar. If still unspecified, the gauge will use two-tone gradiation.
TP_Status_Img "" If Use_Dual_Gradient_Bars is false, specifies the image to be used to fill the TP bar. If still unspecified, the gauge will use two-tone gradiation.
AP_Status_Img "" If Use_Dual_Gradient_Bars is false, specifies the image to be used to fill the AP bar. If still unspecified, the gauge will use two-tone gradiation.

Boolean Values
Option Name Default Description
Always_Allow_Place true When set true, even if 'Actor#' events are present, 'place' events will still function. If false, only 'Actor#' events will function, and the player will not have the option to place any other characters.
Use_Mini_Battle true Instead of evaluating attacks, items or skill usage on the battle map, displays a short scene of the attack, item or skill being applied to the target by the user.
Use_Mini_War true Not currently implemented.
MOVE_CONFIRM true
ATTACK_CONFIRM true
WAIT_CONFIRM true
Force_Wait false After both movement and an action have been made, forces the character into wait phase.
ENABLE_MOVE_START false
SHOW_MOVE_ATTACK true If true, displays the attackable area after having moved.
MOVE_INCLUDE_SPELL false
ACT_LIST true Allows the user to view the list of upcoming turns for actors and enemies.
HIDE_CURSOR_POSITION_INFO false If true, hides the cursor x, y and height from the user.
PREVIEW_DAMAGE true Displays an estimate of the damage a unit will deal when attacking, prior to confirming the target. Does not account for variance, and also displays the chance of hitting. If the attack may apply a state, will display that, but will not show the chance of applying the state.
HIDE_INACTIVE_COMMANDS true
Use_Dual_Gradiant_Bars false Usable, but may cause lag at the end of movement.

Integer Values
Option Name Default Description
SCROLL_SPEED 4 The speed at which the screen will scroll to the current target during battle.
CONTROL_OPACITY 180 Defines the opacity of all command windows in battle.
DIM_OPACITY 120 Used during team mode to mark characters that have already acted
STAT_ANIM_TIME 2 Defines the number of frames increases or decreases in a gauge's value will animate in. Lowest acceptable value is 1. Higher values will lead to minor lag when bars are drawn.
