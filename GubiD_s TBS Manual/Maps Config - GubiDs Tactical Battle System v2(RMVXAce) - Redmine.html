Maps Config¶

Use Alternate Map for Battle¶

By Map¶

battle_on=map_ID

map_ID: integer; ID of map to be used for battles on this map, rather than this map itself. Specifies the default battle map for the whole map.

By Region¶

bat_region[Region_ID]=BATTLE_MAP_ID

Region_ID: integer; ID of region as shown in region-editing mode.
BATTLE_MAP_ID: integer; ID of map to be used for battles triggered when standing on the specified region.

Any regions with set battle maps will override the default battle map for that given map; if battles are triggered on those regions, they will occur on the specified region battle map. But outside of those regions, the default map will be used.

Alternate Exit Map¶

exit_info=[MAPID, X, Y, DIR]

MAPID: integer; ID of map the player should be sent to, following battle.
X: integer; X coordinate
Y: integer; Y coordinate
DIR: integer; direction as on numpad - 2 is down, 4 is left, 8 is up, and 6 is right.

Applies to all battles called from the map with this tag, including battles sent to another battle map.

Backdrop Settings¶

default_back= Backdrop_Name
backdrop[Region_ID] = Backdrop_Name

Region_ID: Number; Used to determine which region this backdrop should be associated with.
Backdrop_Name: string; Used to specify the filename in which should be loaded from the backdrops directory to be displayed during Mini Battle.

If no region backdrop is defined, then it will default to the default_back assigned backdrop.
