Item setup

Put these in the notebox of an item.

Item range

range=[range_max, Field, LINE_ITEM?, exclude_luncher?, range_min, vertical_range, vertical_aoe]

range_max, range_min: numerical; determines the maximum and minimum ranges within which the item can be targeted. 0 is the tile the luncher is standing on.
Field: determines the AoE of the item; 0 would affect only the target tile, whilst 1 would affect directly-adjacent tiles in addition to the target.
LINE_ITEM?: true or false; determines whether the luncher can only target tiles in a horizontal or vertical line from themselves. AoE items can still affect units outside of the line, however.
exclude_luncher?: true or false; prevents the luncher from being affected by the item, even if they're within the AoE.
vertical_range: integer; optional, the number of pixels upwards or downwards from the source tile that the item will accept as a valid target tile. Only used by MGC's isometric battle engine, and only if range_max is more than 1.
vertical_aoe: integer; optional, the number of pixels upwards or downwards from the target tile that the item will apply AoEs to. Only used by MGC's isometric battle engine, and only if Field is 1 or more.
